# `ab-nextjs-theme`

> IMPORTANT: This is a work in progress and subject to major changes until version 1.0.


üé® This is a lightweight collection of client React components as theme abElements created by [Abraham Ukachi](https://github.com/abraham-ukachi), and optimized for [Next.js](https://nextjs.org/docs) applications üòé. 



## Getting Started

### Installation

#### npm

```bash
npm i ab-nextjs-theme 
```

#### pnpm

```bash
pnpm install ab-nextjs-theme 
```
